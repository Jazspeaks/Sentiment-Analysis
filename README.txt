# Sentiment Analysis with Sentiment140

## Overview
This project implements a sentiment analysis model to classify tweets as positive or negative using the Sentiment140 dataset.

## Features
- Data preprocessing: Cleaning and tokenization of text data.
- Model: LSTM neural network for sentiment classification.
- Performance evaluation: Metrics to assess model accuracy.

## Dataset
The dataset used is the [Sentiment140 dataset](https://www.kaggle.com/datasets/kazanova/sentiment140), which contains 1.6 million tweets labeled for sentiment.
